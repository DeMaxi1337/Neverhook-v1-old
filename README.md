## Features

- **Player:** noclip, no death effect, no respawn flash
- **Macros:** nothing
- **Global:** FPS bypass, speedhack
- **Cosmetic:** nothing
- **Bypass:** unlock all icons
- **Creator:** nothing
- **Scripts:** nothing
